# Aprimorando-Conhecimentos :mag_right:

## Tópicos trabalhados neste projeto:
- JSX;
- Components;
- Props;
- Props with type;
- CSS Modules;
- Fragments;
- Events;
- Conditional Rendering;
- Lists Rendering;
- State Lift;

- React Icons.
